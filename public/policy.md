# Politique de Confidentialité

## Projet : Pendu Pokémon (Site Web & Bot Discord)

### Date d’entrée en vigueur : 02 mai 2026

#### 1. Introduction

La présente Politique de confidentialité a pour but d'informer les utilisateurs du site web et du bot Discord "Pendu Pokémon" sur la manière dont leurs données sont collectées, utilisées et protégées. Nous accordons une importance capitale à la protection de votre vie privée.

#### 2. Données collectées

Nous limitons la collecte de données au strict nécessaire pour assurer le fonctionnement du jeu de pendu.

- Via le Bot Discord : Nous collectons uniquement votre ID d'utilisateur Discord (un identifiant numérique unique, ex: 123456789012345678). Nous ne collectons ni votre pseudonyme, ni votre adresse e-mail, ni votre avatar.

- Via le Site Web : Si vous naviguez simplement sur le site, aucune donnée personnelle n'est requise. Si un système de compte existe, seules les données que vous fournissez volontairement sont stockées.

- Données de Jeu : Nous stockons les lettres que vous proposez durant une partie afin de calculer votre progression.

#### 3. Finalité du traitement

Les données collectées servent exclusivement à :

- Identifier votre partie en cours afin que vous puissiez la reprendre là où vous l'avez laissée.

- Assurer le bon fonctionnement des commandes /pendu et /deviner.

- (Optionnel) Établir des statistiques anonymes sur le nombre de parties jouées.

#### 4. Conservation des données

Nous pratiquons une politique de conservation minimale :

- Parties actives : Les données de jeu liées à votre ID Discord sont conservées en base de données jusqu'à ce que la partie soit terminée (victoire ou échec).

- Nettoyage automatique : Toute partie inactive depuis plus de 24 heures est susceptible d'être supprimée automatiquement de notre base de données (via un script de nettoyage).

#### 5. Partage des données

Aucune donnée n'est vendue, louée ou partagée avec des tiers. Les données résident sur les serveurs de notre hébergeur (OVH, France) et ne sont accessibles qu'à l'administrateur technique du projet. Notez que Discord a accès aux interactions que vous effectuez sur leur plateforme conformément à leur propre politique de confidentialité.

#### 6. Vos Droits (RGPD)

Conformément à la réglementation européenne, vous disposez des droits suivants :

- Droit d'accès et de rectification : Vous pouvez demander quelles données sont stockées vous concernant.

- Droit à l'effacement : Vous pouvez demander la suppression de votre partie en cours.

- Droit d'opposition : En cessant d'utiliser les commandes du bot, aucune nouvelle donnée ne sera collectée.

Pour exercer ces droits, vous pouvez contacter l'administrateur via le serveur Discord ou par les moyens de contact mis à disposition sur le site web.

#### 7. Sécurité

Nous mettons en œuvre des mesures de sécurité techniques (chiffrement SSL sur le site, vérification des signatures cryptographiques des webhooks Discord) pour protéger vos données contre tout accès non autorisé.
